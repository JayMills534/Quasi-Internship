Proposal (In-progress):
The hybrid quantum PINN, as exemplified by the Cz growth example, uses a loss function based on integer PDEs suitable for specific physical conditions [1].
A Quadrature Enhanced Monte Carlo fractional PINN (QE-MC-fPINN) provides a different loss function to implement to account for physical conditions best expressed
with fractional PDEs [2]. Thus, implementing this change to the loss/residual part of the HQ-PINN may provide advantages in versatility for fluid flow in porous media.
